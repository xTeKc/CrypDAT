### Crypto DATA Grab.

The goal of this project is to get all data for any cryptocurrency on any blockchain, including the smart contract source code with syntax highlighting.

##### Blockchains Supported:
+ Cardano
+ Ethereum
+ Binance
+ Fantom
+ Harmony Shard
+ Huobi
+ Polygon
+ Solana  
+ xDai

##### Contracts Sorted by Language:
+ Solidity
+ Plutus
+ Vyper
